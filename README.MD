How To Activate The App:

Server-side

1. cd server
2. run "npm i" in the terminal
3. run "touch .env" and fill in the env just like .env copy
4. run "nodemon app" (if you don't have nodemon try npm i -G nodemon)

Admin Side

1. cd admin
2. cd admin-side
3. run "npm i"
4. run "npm run dev"

Client Side

1. cd front
2. cd client-side
3. run "npm i"
4. run "npm run dev"
